## Quick Install

Install HeroUI and required dependencies:

<Tabs items={["npm", "pnpm", "yarn", "bun"]}>
  <Tab value="npm">
    ```bash
    npm install @heroui/react@alpha
    ```
  </Tab>
  <Tab value="pnpm">
    ```bash
    pnpm add @heroui/react@alpha
    ```
  </Tab>
  <Tab value="yarn">
    ```bash
    yarn add @heroui/react@alpha
    ```
  </Tab>
  <Tab value="bun">
    ```bash
    bun add @heroui/react@alpha
    ```
  </Tab>
</Tabs>

## Import Styles

Add to your main CSS file:

```css
@import "tailwindcss";
@import "@heroui/styles";
```

<Callout type="warning">
  Import order matters. Always import `tailwindcss` first.
</Callout>

## Use Components

```tsx
import { Button } from '@heroui/react';

function App() {
  return (
    <Button>
      My Button
    </Button>
  );
}
```
