### Anatomy

Import all parts and piece them together.

```tsx
import { Alert } from '@heroui/react';

export default () => (
  <Alert>
    <Alert.Icon />
    <Alert.Content>
      <Alert.Title>Alert title</Alert.Title>
      <Alert.Description>
        Alert description with more details
      </Alert.Description>
    </Alert.Content>
    <Alert.Action>Action</Alert.Action>
    <Alert.Close />
  </Alert>
)
```

## Styling

### Passing Tailwind CSS classes

```tsx
import { Alert } from '@heroui/react';

function CustomAlert() {
  return (
    <Alert className="border-2 border-blue-500">
      <Alert.Icon className="text-blue-600" />
      <Alert.Content>
        <Alert.Title className="font-bold">Custom styled alert</Alert.Title>
        <Alert.Description className="text-sm">
          With custom Tailwind classes
        </Alert.Description>
      </Alert.Content>
    </Alert>
  );
}
```

### CSS Classes

The Alert component uses these CSS classes:

#### Base Classes
- `.alert` - Base alert styles
- `.alert__icon` - Icon container styles
- `.alert__content` - Content wrapper styles
- `.alert__title` - Title text styles
- `.alert__description` - Description text styles
- `.alert__action` - Action button styles
- `.alert__close` - Close button styles

#### Variant Classes
- `.alert--default` - Default variant
- `.alert--success` - Success variant
- `.alert--warning` - Warning variant
- `.alert--danger` - Danger variant

## Accessibility

#### Keyboard Navigation

- **Tab** - Move focus between interactive elements (action and close buttons)
- **Enter/Space** - Activate focused button

#### Screen Reader Support

- Alert content is announced when it appears on screen
- Action and close buttons are properly labeled
- Variant states are communicated through appropriate ARIA attributes
