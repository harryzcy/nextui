## Installation

```bash
npm install @heroui/react tailwindcss
```

Note: HeroUI v3 requires Tailwind CSS v4. See the [Quick Start](/docs/overview/quick-start) guide for setup instructions.

## Import

```tsx
import { Avatar } from '@heroui/react';
```

## Usage

### Basic Avatar

The Avatar component uses a compound pattern with `Avatar.Image` and `Avatar.Fallback` parts.

<ComponentPreview>
  <AvatarBasic />

```tsx
</Avatar>
```
</ComponentPreview>

### Fallback with Icons

Use icons as fallback content when no initials are available.


```tsx
import { Icon } from '@iconify/react';

<div className="flex gap-4">
  <Avatar>
    <Avatar.Fallback>
      <Icon icon="gravity-ui:person" />
    </Avatar.Fallback>
  </Avatar>
  <Avatar>
    <Avatar.Fallback>
      <Icon icon="gravity-ui:person-gear" />
    </Avatar.Fallback>
  </Avatar>
</div>
```

### Delayed Fallback

Control when the fallback appears using the `delayMs` prop. This prevents flickering when images load quickly.


```tsx
<Avatar>
  <Avatar.Image
    src="https://img.heroui.chat/image/avatar?w=400&h=400&u=4"
    alt="User"
  />
  <Avatar.Fallback delayMs={600}>JG</Avatar.Fallback>
</Avatar>
```

## Compound Parts

### Avatar (Root)

The root container that manages the avatar state and provides context to child components.

### Avatar.Image

Displays the user's profile image. If the image fails to load, the fallback will be shown.

### Avatar.Fallback

Content to display when the image is unavailable or still loading. Can contain text, initials, or icons.

## Accessibility

### Image Alt Text

Always provide meaningful alt text for avatar images:

```tsx
<Avatar>
  <Avatar.Image
    src="user-avatar.jpg"
    alt="John Doe's profile picture"
  />
  <Avatar.Fallback>JD</Avatar.Fallback>
</Avatar>
```

### Fallback Content

Ensure fallback content is meaningful:

```tsx
// Good: Meaningful initials
<Avatar.Fallback>JD</Avatar.Fallback>

// Good: Descriptive icon
<Avatar.Fallback>
  <Icon icon="gravity-ui:person" aria-label="User avatar" />
</Avatar.Fallback>

// Avoid: Generic symbols
<Avatar.Fallback>?</Avatar.Fallback>
```

## API Reference

### Avatar Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `size` | `'sm' \| 'md' \| 'lg'` | `'md'` | Size of the avatar |
| `className` | `string` | - | Additional CSS classes |
| `children` | `React.ReactNode` | - | Avatar.Image and Avatar.Fallback components |

### Avatar.Image Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `src` | `string` | - | Image source URL |
| `alt` | `string` | - | Alternative text for the image |
| `className` | `string` | - | Additional CSS classes |
| `onLoadingStatusChange` | `(status: 'idle' \| 'loading' \| 'loaded' \| 'error') => void` | - | Callback for loading status changes |

### Avatar.Fallback Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `delayMs` | `number` | `0` | Delay in milliseconds before showing fallback |
| `className` | `string` | - | Additional CSS classes |
| `children` | `React.ReactNode` | - | Fallback content (text, initials, or icons) |

## Styling

### CSS Classes

The Avatar component uses these CSS classes:

- `.avatar` - Base avatar styles
- `.avatar--{size}` - Size-specific styles
- `.avatar__image` - Image element styles
- `.avatar__fallback` - Fallback element styles

### Custom Styling

```tsx
// Custom avatar with different styling
<Avatar className="border-2 border-blue-500 shadow-lg">
  <Avatar.Image src="..." alt="Custom styled avatar" />
  <Avatar.Fallback className="bg-blue-100 text-blue-700">
    CS
  </Avatar.Fallback>
</Avatar>
```

### Size Customization

```css
/* Custom avatar size */
.avatar-xl {
  width: 5rem;
  height: 5rem;
}

.avatar-xl .avatar__fallback {
  font-size: 1.5rem;
}
```

## Examples

### User Profile Card

```tsx
import { Avatar, Card } from '@heroui/react';

function UserProfile({ user }) {
  return (
    <Card>
      <Card.Content className="flex items-center gap-4 p-6">
        <Avatar size="lg">
          <Avatar.Image src={user.avatar} alt={`${user.name}'s avatar`} />
          <Avatar.Fallback>
            {user.name.split(' ').map(n => n[0]).join('')}
          </Avatar.Fallback>
        </Avatar>
        <div>
          <h3 className="font-semibold">{user.name}</h3>
          <p className="text-gray-600">{user.role}</p>
          <p className="text-sm text-gray-500">{user.email}</p>
        </div>
      </Card.Content>
    </Card>
  );
}
```


### Avatar with Status Indicator

```tsx
function AvatarWithStatus({ user, status }) {
  const statusColors = {
    online: 'bg-green-500',
    offline: 'bg-gray-400',
    busy: 'bg-red-500',
    away: 'bg-yellow-500'
  };

  return (
    <div className="relative">
      <Avatar>
        <Avatar.Image src={user.avatar} alt={user.name} />
        <Avatar.Fallback>{user.initials}</Avatar.Fallback>
      </Avatar>
      <div 
        className={`absolute -bottom-0.5 -right-0.5 w-3 h-3 rounded-full border-2 border-white ${statusColors[status]}`}
        aria-label={`User is ${status}`}
      />
    </div>
  );
}
```
