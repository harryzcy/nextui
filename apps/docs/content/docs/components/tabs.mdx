### CSS Classes

The Tabs component uses these CSS classes:

#### Base Classes
- `.tabs` - Base tabs container
- `.tabs__list` - Tab list container
- `.tabs__tab` - Individual tab button
- `.tabs__panel` - Tab panel content

#### Variant Classes
- `.tabs--default` - Default variant with surface styling
- `.tabs--line` - Line variant with underline indicator

#### Orientation Classes
- `.tabs--horizontal` - Horizontal tab layout (default)
- `.tabs--vertical` - Vertical tab layout

### Interactive States

The component supports both CSS pseudo-classes and data attributes:

- **Selected**: `[aria-selected="true"]`
- **Hover**: `:hover` or `[data-hover="true"]`
- **Focus**: `:focus-visible` or `[data-focus-visible="true"]`
- **Disabled**: `[aria-disabled="true"]`

## Accessibility

#### Keyboard Navigation

- **Tab** - Move focus to the next focusable element
- **Shift + Tab** - Move focus to the previous focusable element
- **Arrow Left/Right** - Navigate between tabs (horizontal orientation)
- **Arrow Up/Down** - Navigate between tabs (vertical orientation)
- **Home** - Move focus to the first tab
- **End** - Move focus to the last tab
- **Enter/Space** - Select the focused tab

#### Screen Reader Support

- Tab list is announced with its label
- Selected state is announced for active tab
- Tab panels are associated with their tabs via ARIA attributes
- Disabled tabs are properly announced

#### Focus Management

- Focus moves to tab panel content when a tab is selected
- Focus trap ensures keyboard navigation stays within tabs
- Focus visible indicators for keyboard navigation

## API Reference

### Tabs Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `variant` | `"default" \| "line"` | `"default"` | Visual style variant |
| `orientation` | `"horizontal" \| "vertical"` | `"horizontal"` | Tab layout orientation |
| `selectedKey` | `string` | - | Controlled selected tab key |
| `defaultSelectedKey` | `string` | - | Default selected tab key |
| `onSelectionChange` | `(key: string) => void` | - | Selection change handler |
| `className` | `string` | - | Additional CSS classes |

### Tabs.List Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `aria-label` | `string` | - | Accessibility label for tab list |
| `className` | `string` | - | Additional CSS classes |

### Tabs.Tab Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `id` | `string` | - | Unique tab identifier |
| `isDisabled` | `boolean` | `false` | Whether tab is disabled |
| `className` | `string` | - | Additional CSS classes |

### Tabs.Panel Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `id` | `string` | - | Panel identifier matching tab id |
| `className` | `string` | - | Additional CSS classes |
