### Anatomy

Import all parts and piece them together.

```tsx
import { Checkbox, Label, Description } from '@heroui/react';

// Basic checkbox with external label
export default () => (
  <div className="flex items-center gap-3">
    <Checkbox id="terms">
      <Checkbox.Indicator />
    </Checkbox>
    <Label htmlFor="terms">Accept terms</Label>
  </div>
)

// Checkbox with description
export const WithDescription = () => (
  <div className="flex gap-3">
    <Checkbox className="mt-0.5" id="notifications">
      <Checkbox.Indicator />
    </Checkbox>
    <div className="flex flex-col gap-1">
      <Label htmlFor="notifications">Notifications</Label>
      <Description>Get email alerts</Description>
    </div>
  </div>
)

// CheckboxGroup for multiple options
export const Group = () => (
  <CheckboxGroup>
    <Label>Select options</Label>
    <CheckboxGroup.Items>
      <div className="flex items-center gap-3">
        <Checkbox value="option1" id="opt1">
          <Checkbox.Indicator />
        </Checkbox>
        <Label htmlFor="opt1">Option 1</Label>
      </div>
    </CheckboxGroup.Items>
  </CheckboxGroup>
)
```

## Styling

### Passing Tailwind CSS classes

```tsx
import { Checkbox, Label } from '@heroui/react';

function CustomCheckbox() {
  return (
    <div className="flex items-center gap-3">
      <Checkbox 
        id="custom"
        className="border-2 border-blue-500"
      >
        <Checkbox.Indicator className="text-blue-600" />
      </Checkbox>
      <Label htmlFor="custom" className="text-lg font-semibold">
        Custom styled checkbox
      </Label>
    </div>
  );
}
```

### CSS Classes

The Checkbox component uses these CSS classes:

#### Base Classes
- `.checkbox` - Base checkbox container
- `.checkbox__indicator` - Check indicator element
- `.checkbox__icon` - Icon inside indicator

#### State Classes
- `.checkbox--selected` - Checked state
- `.checkbox--indeterminate` - Indeterminate state
- `.checkbox--disabled` - Disabled state
- `.checkbox--invalid` - Invalid/error state
- `.checkbox--required` - Required field state

#### CheckboxGroup Classes
- `.checkbox-group` - Group container
- `.checkbox-group--horizontal` - Horizontal layout
- `.checkbox-group--vertical` - Vertical layout
- `.checkbox-group__items` - Items wrapper

### Interactive States

The component supports both CSS pseudo-classes and data attributes:

- **Checked**: `:checked` or `[data-selected="true"]`
- **Indeterminate**: `[data-indeterminate="true"]`
- **Hover**: `:hover` or `[data-hover="true"]`
- **Focus**: `:focus-visible` or `[data-focus-visible="true"]`
- **Disabled**: `:disabled` or `[aria-disabled="true"]`
- **Invalid**: `[aria-invalid="true"]`

## Accessibility

#### Keyboard Navigation

- **Tab** - Move focus to the checkbox
- **Space** - Toggle checkbox state
- **Arrow keys** - Navigate between checkboxes in a group

#### Screen Reader Support

- Checkbox state is announced (checked/unchecked/mixed)
- Labels are properly associated with checkboxes
- Required and invalid states are communicated
- Group role and label are announced for checkbox groups

#### Focus Management

- Clear focus indicators for keyboard navigation
- Focus moves naturally through form elements
- Group navigation with arrow keys

## API Reference

### Checkbox Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `isSelected` | `boolean` | `false` | Whether checkbox is checked |
| `defaultSelected` | `boolean` | `false` | Default checked state |
| `isIndeterminate` | `boolean` | `false` | Indeterminate state |
| `isDisabled` | `boolean` | `false` | Whether checkbox is disabled |
| `isInvalid` | `boolean` | `false` | Whether checkbox is invalid |
| `isRequired` | `boolean` | `false` | Whether checkbox is required |
| `value` | `string` | - | Value for form submission |
| `onChange` | `(isSelected: boolean) => void` | - | Change handler |
| `id` | `string` | - | HTML id attribute |
| `className` | `string` | - | Additional CSS classes |

### Checkbox.Indicator Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `className` | `string` | - | Additional CSS classes |
| `children` | `ReactNode` | - | Custom check icon |

### CheckboxGroup Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `value` | `string[]` | - | Selected values |
| `defaultValue` | `string[]` | - | Default selected values |
| `onChange` | `(value: string[]) => void` | - | Selection change handler |
| `orientation` | `"horizontal" \| "vertical"` | `"vertical"` | Layout orientation |
| `isDisabled` | `boolean` | `false` | Disable all checkboxes |
| `isInvalid` | `boolean` | `false` | Invalid state for group |
| `isRequired` | `boolean` | `false` | Required state for group |
| `className` | `string` | - | Additional CSS classes |

### CheckboxGroup.Items Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `className` | `string` | - | Additional CSS classes |
