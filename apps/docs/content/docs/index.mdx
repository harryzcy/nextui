## Philosophy

HeroUI v3 embraces the compound component pattern, giving you complete control over markup structure while providing robust accessibility features through React Aria Components. Each component is broken down into composable parts that work together seamlessly.

```tsx
// HeroUI v2 - Monolithic approach
import {Accordion, AccordionItem} from "@heroui/react";

<Accordion>
  <AccordionItem key="1" aria-label="Accordion 1" title="Accordion 1">
    Lorem ipsum dolor sit amet...
  </AccordionItem>
  <AccordionItem key="2" aria-label="Accordion 2" title="Accordion 2">
    Lorem ipsum dolor sit amet...
  </AccordionItem>
</Accordion>

// HeroUI v3 - Compound component approach
import { Accordion } from "@heroui/react";

<Accordion>
  <Accordion.Item>
    <Accordion.Heading>
      <Accordion.Trigger>
        Accordion 1
        <Accordion.Indicator />
      </Accordion.Trigger>
    </Accordion.Heading>
    <Accordion.Panel>
      <Accordion.Body>
        Lorem ipsum dolor sit amet...
      </Accordion.Body>
    </Accordion.Panel>
  </Accordion.Item>
</Accordion>
```
