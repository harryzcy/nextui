## Version 3.0.0-alpha.12 (Latest)

*Released: January 2025*

### New Features

- ✨ **Default Size Pattern** - All components now include default sizes in base classes
- 🎨 **Enhanced Styling System** - Improved CSS-based styling with BEM conventions
- 📦 **Component Updates** - Updated exports and improved TypeScript definitions

### Components

- Updated all components to follow default size pattern
- Enhanced component exports for better tree-shaking
- Improved TypeScript definitions and prop interfaces

### Bug Fixes

- Fixed component appearance issues when no size variant is provided
- Resolved styling inconsistencies across components
- Improved accessibility attributes and ARIA support

### Breaking Changes

- Component styling now follows BEM naming convention
- Some prop names have been standardized across components
- CSS class structure has been updated for consistency

## Migration Guide

### From Alpha 11 to Alpha 12

1. **Update Dependencies**
   ```bash
   npm install @heroui/react@3.0.0-alpha.12
   ```

2. **CSS Classes**
   - Component classes now follow BEM convention
   - Update custom styles to use new class names
   - Default sizes are now included in base classes

3. **Component Props**
   - Some prop names have been standardized
   - Check TypeScript errors for updated prop interfaces
   - Update component compositions if needed

### From v2 to v3

v3 is a complete rewrite with breaking changes. See our [Migration Guide](/docs/guides/migration) for detailed instructions.

## Release Schedule

HeroUI v3 follows semantic versioning and has a regular release cycle:

- **Alpha Releases** - Weekly during development
- **Beta Releases** - Monthly with feature freeze
- **Stable Releases** - Quarterly with full testing
