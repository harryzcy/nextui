## What's New

### RAC Upgrade

The React Aria Components versions have been upgraded to the latest [October 2, 2025 Release](https://react-spectrum.adobe.com/releases/2025-10-02.html).

This upgrade introduces two CSS variables `--disclosure-panel-width` and `--disclosure-panel-height` in [Disclosure](/docs/components/disclosure) and [Disclosure Group](/docs/components/disclosure-group), which are set to the scrollWidth and scrollHeight of the panel when expanding and collapsing.

### Tabs Indicator Redesign

The [Tabs component](/docs/components/tabs) has been updated with a new structure that leverages React Aria's `SelectionIndicator` internally, and now supports server-side rendering (SSR).

**Breaking Changes:**

- Moved `Tabs.Indicator` inside each `Tabs.Tab`

**Before:**
```tsx
<Tabs>
  <Tabs.ListWrapper>
    <Tabs.List aria-label="Options">
      <Tabs.Tab/>
    </Tabs.List>
    <Tabs.Indicator />
  </Tabs.ListWrapper>
  <Tabs.Panel/>
</Tabs>
```

**After:**
```tsx
<Tabs>
  <Tabs.ListWrapper>
    <Tabs.List aria-label="Options">
      <Tabs.Tab>
        <Tabs.Indicator />
      </Tabs.Tab>
    </Tabs.List>
  </Tabs.ListWrapper>
  <Tabs.Panel/>
</Tabs>
```

The [Switch component](/docs/components/switch) has been updated with a new variant `size`. The available options are `sm`, `md` and `lg`.
